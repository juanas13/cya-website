# Site CYA - Landing Page Officielle du Token CryptoIA

Site web pour le token CYA (ERC20 upgradeable sur BSC) avec features staking, rewards IA et gouvernance.

## Aperçu
- **Tech Stack** : Next.js, TypeScript, Tailwind CSS.
- **Features** : Connexion wallet, dashboard staking, mint rewards IA.
- **Live** : https://cya-site.vercel.app/
## Installation Locale
1. Clonez : `git clone https://github.com/juanas13/cya-website.git`
2. Installez : `cd cya-website && npm install`
3. Lancez : `npm run dev` (sur http://localhost:3000)
4. Build : `npm run build` pour tester prod.

## Scripts (dans package.json)
- `dev` : Développement.
- `build` : Build prod.
- `start` : Serveur prod.
